# Code of Conduct

MacPad Pro follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful, constructive, and direct. Harassment, personal attacks, and abusive behavior are not acceptable in issues, pull requests, discussions, or other project spaces.

Report unacceptable behavior by opening a private security advisory or contacting the project owner through GitHub.
